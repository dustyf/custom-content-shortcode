Adds a shortcode to get content or field from any post type by name. For example: [custom type="room" name="room-name" field="rent"]

== Description ==

Adds a shortcode to get the content or field from a post/page/custom post type, specified by type and name. For example: [custom type="room" name="room-name" field="rent"]

Post type can be "post", "page", or a custom post type.  Name is the slug of the post.  If no field is specified, it returns the content of the post.

Additionally, you can use the following field names:

* "title" - return the title</li>
* "link" - return the link url</li>
* "thumb" - return the url of the featured image</li></ul>

More may be added as the need arises.

== Installation ==

1. Upload `custom-content-shortcode.zip` through Plugins->Add New, or extract and upload the folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place the shortcode in a post, page, etc.
