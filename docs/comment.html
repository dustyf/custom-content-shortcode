Comments

Use the [comments] shortcode to display recent comments, or those of the current post.

Display the 5 most recent comments

[comments count="5"]     [comment content] by [comment author] [/comments]

Parameters

Available parameters for the [comments] shortcode are:
  • count - number of comments to show, or set to all
  • id - set to this to display comments from the current post only
  • format - set to false to prevent formatting comment content
  • words/length - trim by number of words or characters (for comment content or post title)
  • date_format - format the comment date, e.g., "Y-m-d". Codex: Formatting Date and Time

Fields

Available [comment] fields:
  • content - content of the comment
  • date - date of comment
  • title - title of the post where the comment is
  • url - URL of the post
  • title-link - post title with link to the post
  • author - author name
  • author-url - author URL
  • author-link - author name with link to URL; if no URL, then displays just name

Comment list, input form and total count

You can also display comments of the current post using the theme's comment template.

Display comment list using template

Comment list: [comment template] or [comments] By default, the comments list is displayed by comments.php in the theme directory. If you want to specify a different template: [comment template="short-comments.php"] WordPress will look for the comments template in the child theme first (if it exists), then in the parent theme.

Display comment input form

[comment form]

Display number of comments for the current post

Number of comments: [comment count] You can also display the total comment count after a loop is finished: Total number of comments: [comment total]

Conditional: if current post has comment

Use [if comment] to display something if the current post has any comments or not.

Display comment count if the post has comment

[loop type="post" category="news"]     [if comment]         [comment count]     [/if]     [if not comment]         No comment yet.     [/if] [/loop]
