Conditionals

With the [if] shortcode, you can display content based on certain conditions. [loop type="films"]     [content field="title"]     [if not category="drama"]         [content taxonomy="category"]     [/if]     [if category="drama"]         DRAMA!!     [/if] [/loop]

Parameters

The available parameters are:
  • type - post type
  • name - post name/slug

Category, tag, taxonomy

  • category - if post has category
  • tag - if post has tag
  • taxonomy - name of custom taxonomy to query
  • term - if post has term(s) in the specified taxonomy
You can pass multiple values, for example: category="sci-fi,comedy". This returns posts in either Sci-Fi or Comedy. If you want only posts matching both categories, use the compare parameter:
  • compare - Set compare="AND" to query for all terms.

Template conditionals

You can also display depending on template status. For now, there is only:
  • home - front page
It's used without parameter value. [if home]     Home sweet home. [/if]
