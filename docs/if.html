Conditionals

With the [if] shortcode, you can display content based on certain conditions. [loop type="films"]     [field title]     [if not category="drama"]         [content taxonomy="category"]     [/if]     [if category="drama"]         DRAMA!!     [/if] [/loop]

Parameters

The available parameters are:
  • type - post type
  • name - post name/slug

Category, tag, taxonomy

  • category - if post has category
  • tag - if post has tag
  • taxonomy - name of taxonomy to query
  • term - if post has term(s) in the specified taxonomy

Field value

  • field - name of field to query
  • value - if post has value(s) in the specified field
    • You can pass multiple values, for example: category="sci-fi, comedy". This returns posts in either Sci-Fi or Comedy. If you want only posts matching both categories, use the compare parameter.
  • compare - Set compare="AND" to query for all terms.

Other conditions

  • flag - if custom field value exists; for example: flag="field_name"
  • attached - if the post has any attachments
  • empty - if the loop is empty (no post found)
  • first, last - if it's the first or last post found
  • every - for every X post in the loop
  • not - when the condition is not met

Conditional: if there is no query result

Use [if empty] to display something when there is no post matching the query.

Display a message for no query result

[loop type="events" category="weekend"]     [field title]     [field description]     [if empty]         There is no event for this category.     [/if] [/loop] If there's no post found, the loop displays only what's inside [if empty].

Conditional: for every X number of posts

Add an extra break line for every 3 posts

[loop type="post" count="9"]     [field title]     [if every="3"]         <br>     [/if] [/loop]

Display something for the first or last post

[loop type="post"]     [field title]     [if first]         (first post)     [/if]     [if last]         (last post)     [/if] [/loop]

Conditional: if a field value exists

Display only products that have serial numbers

[loop type="product"]     [if flag="serial_number"]         Product: [field title]         Serial #: [flag]     [/if] [/loop] The flag parameter specifies the field, and its value can be displayed with the [flag] shortcode. In addition, you can display something when a field is empty: [loop type="product"]     Product: [field title]     [if flag="serial_number"]         Serial #: [flag]     [/if]     [if not flag="serial_number"]         No Serial Number     [/if] [/loop]

Current template

You can also display something depending on current template. For now, there is only:
  • home - front page (recent posts or static page)
It's used without parameter value. [if home]     Home sweet home. [/if]
