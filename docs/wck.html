WCK Fields and Post Types

Use [wck-field] to display a field from a metabox.

Display fields

[wck-field meta="metabox_name" name="field_name"] [wck-field meta="another_metabox" name="another_field"]

Parameters

  • id - post id (default is current post)
  • meta - name of metabox
  • name - name of field

Shortcode in field

  • shortcode - set true to enable shortcodes in the field
Usually WCK processes a text area to create break lines, but it doesn't work well with shortcodes. When shortcodes are enabled, you'll need to insert <br> tags manually.

Repeater

Use [wck-repeat] to go through repeater fields.

Display repeater fields

[wck-repeat meta="metabox_name"]     [wck-field name="field_name"]     [wck-field name="another_field"] [/wck-repeat]

Parameters

  • meta - name of metabox
  • id - post id (default is current post)

Loop

You can use these inside [loop] to display fields from multiple posts.

Display repeater fields from five recent posts

[loop type="post_type" count="5"]     Post Title: [field title]     [wck-repeat meta="metabox_name"]         [wck-field name="field_name"]         [wck-field name="another_field"]     [/wck-repeat] [/loop]
