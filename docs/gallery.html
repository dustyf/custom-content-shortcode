Gallery field

In the admin menu, under Settings -> Gallery Fields, there is an option to enable a simple gallery field for any post type. Images can be added, removed, and ordered.

Display image details from the gallery field of a specific post

[loop field="gallery" name="hello-world"]     Title: [field title]     Full-size image: [field image]     Caption: [field caption] [/loop] The [loop] shortcode handles the gallery field just like attachments. For each image, the [content] or [field] shortcode can display these fields: id, title, image, image-url, caption, description, thumbnail, and thumbnail-url. The [content] shortcode can display individual images of the gallery field.

Display the 3rd image in the gallery field

[content field="gallery" num="3"] ..or.. [field gallery num="3"] It can also display all images in the field using a native or Bootstrap gallery.

Display the images in a native gallery or a Bootstrap carousel

[content gallery="native"] [content gallery="carousel"] You can pass the following parameters to the native gallery: orderby, order, columns, size, link, include, exclude. For details, please refer to the Codex: Gallery Shortcode. When using the [loop] to generate multiple Bootstrap carousels, the following will put images from each post in its own carousel. [content gallery="carousel" group="gallery-{ID}"]

Attachments

For post attachments such as images, you can query the post type attachment and display the necessary fields.

Display details on all the attachments of the current post

[loop type="attachment"]     Attachment ID: [field id]     Title: [field title]     Full-size image: [field image]     Full-size image URL: [field image-url]     Caption: [field caption]     Description: [field description]     Thumbnail: [field thumbnail]     Thumbnail URL: [field thumbnail-url] [/loop]

Display attachment thumbnails of all posts in the category tree, linked to full-sized images in a lightbox

[loop type="attachment" category="tree"]     <a href='[field image-url]' rel="lightbox">         <img src='[field thumbnail-url]'>     </a> [/loop]
