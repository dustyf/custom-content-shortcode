Main Feature

The [content] shortcode displays the following: posts, pages, custom post types, fields, images, attachments, menus, or widget areas. The [loop]Â shortcode performs query loops for a group of content. It can show, for example, available products in a category, or excerpts from the 5 most recent posts.

Included

  • Simple gallery field for any post type
  • Relative URLs for links and images
  • Comments list, input form or comment count
  • User name, ID, login/logout links with redirect
  • Display content for admin, specific user, login status
  • Include HTML/CSS/PHP/JavaScript files anywhere
  • Display content according to Mobile Detect

With other libraries

  • Bootstrap carousel, navbar menu, pills and stacked
  • Advanced Custom Fields - image, gallery, repeater
  • Live Edit - front-end editing of content and fields
