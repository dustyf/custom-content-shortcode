Main Feature

The [content] shortcode displays the following: posts, pages, custom post types, fields, images, attachments, menus, or widget areas. The [loop]Â shortcode performs query loops for a group of content. It can show, for example, available products in a category, or excerpts from the 5 most recent posts.

Additional Features

  • A simple gallery field
  • Relative URLs for links and images
  • Comments list, input form or comment count
  • User name, ID, login/logout links with user redirect
  • Load content by user status: admin, non-admin, logged in/out, or user name
  • Include HTML/CSS/JavaScript

With other libraries

  • Display a Bootstrap carousel, navbar menu, pills and stacked
  • Load content based on device type using WP Mobile Detect
  • Display Advanced Custom Fields - image, gallery, repeater
  • Include Live Edit - front-end editing of content and fields
