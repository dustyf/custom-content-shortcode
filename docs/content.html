Basic Examples

Here are some examples of using the [content] shortcode.

Display post content by name

[content name="hello-world"]

Display the featured image of a page

[content type="page" name="about-me" field="image"]

Display a custom field from a custom post type

[content type="apartment" name="lux-suite-22" field="rent-per-day"]

Display fields from the current post

The Article <b>[content field="title"]</b> by [content field="author"] was written on [content field="date"].

Parameters

The main parameters of the [content] shortcode are:
  • type - which post type to target: post, page, or custom post type - the default is any post type
  • name, title, or id -  which post to target: name/slug, title, or ID - the default is the current post
  • field - which field to display - the default is the main content of the post You can display custom fields as well as predefined fields: id, url, edit-url, title, title-link (title wrapped in a link to the post), date, modified, author, author-id, author-url, avatar, image, image-url, thumbnail, and excerpt.
    • You can also use [field field_name] instead of [content field="field_name"].

Other field types

  • taxonomy - display post taxonomy, such as: category, post_tag, or a custom taxonomy; to output the slug, add: out="slug"
  • meta - display author meta: field="author" meta="user_email" - see: Codex: Author Meta
  • out="label" - when displaying checkboxes or select fields made in ACF, this parameter displays their labels instead of values

Formatting

  • words or length - limit number of words or characters
  • class - add a <div> class to the output for styling purpose
  • format - add paragraph tags <p> and preserve line breaks <br> - set to true/false - by default, post content is formatted and for all other fields, it's set to false
  • embed - auto-embed URLs like YouTube, Vimeo, etc - set to true/false - by default, such URLs in post content are auto-embedded
  • date_format - use a custom format when displaying field="date", "modified", or a custom field For example, "d.m.Y" will display as 18.11.2013. Use double backslashes to escape characters - "Y/m/d \\a\\t g:i A" will show as 2013/11/17 at 11:06 PM. Codex: Formatting Date and Time To format a unix timestamp value, use: in="timestamp"
Read more
  • more - display content up to the <!--more--> tag; more="true" will show "Read More" at the end, with a link to the post; to change the text, use: more="..."
  • link - set link="false" to disable link to the post
Image field
  • image - display an image field
  • size - size of image to get: thumbnail, middle, large, full (default) or custom defined size
  • in - type of image field: id (default), url, or object
Other content types
  • area or sidebar - display a widget area/sidebar by title
  • menu - display a menu list by name/slug, title, or ID. Use the ul parameter for Bootstrap nav-tabs, nav-pills and nav-pills-stacked. For a Bootstrap navbar menu, please see the section for the [navbar] shortcode, under Etc.
