For/each

This is a feature to create a loop for each category, tag, or taxonomy term.

Display 3 most recent posts for each category

[for each="category"]     Category: [each name]     [loop type="post" count="3"]         [field title-link]     [/loop] [/for] It gets all existing terms of a given taxonomy, in this case category, and loops through each term.

Parameters

Available parameters for the [for] shortcode are:
  • each - category, tag, or custom taxonomy
  • count - limit number of terms: count="3"
  • parent - get child categories by parent slug
  • orderby - order by name (default), count (post count), id, or slug
  • order - ASC (ascending - default), or DESC (descending)
  • trim - set true to trim space or comma at the end
Available parameters for the [each] shortcode are:
  • name - name of category, tag, or taxonomy term
  • slug - slug of category, tag, or taxonomy term
  • id - ID of category, tag, or taxonomy term

Loop

It's possible to use [for] and [each] inside a loop.

Display a link for each category assigned to a post

[loop type="post"]     [field title]     [for each="category"]         <a href="[url]/category/[each slug]">[each name]</a>     [/for] [/loop]

Current post

Outside the loop, use the current parameter to display category/tag/taxonomy of the current post only. Otherwise, [for] displays all terms that exist.

Display a link for each category assigned to the current post

[for each="category" current="true"]     <a href="[url]/category/[each slug]">[each name]</a> [/for]

List

You can use the trim parameter to create a list of terms. It removes extra space or comma at the end.

Display a list of categories

[for each="category" trim="true"]     [each name], [/for]

Trim other characters

[for each="category" trim="/"]     [each name] / [/for]
