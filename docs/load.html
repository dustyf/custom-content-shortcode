Load

Use the [load] shortcode to include a file into a page.

Load a sidebar template

[load file="template/sidebar.html"] The file can contain HTML, PHP, CSS, JavaScript, and shortcodes.

Parameters

  • format - set true to format the file with line breaks and paragraph tags
  • shortcode - set false to disable shortcodes
  • dir - load from a directory
    • web - http://
    • site - site address
    • wordpress - WordPress directory
    • content - /wp-content/
    • theme - /wp-content/theme
    • child - /wp-content/child_theme
    • views - /wp-content/views/
By default, it looks for the file in the current theme directory.

Load a file from a different location

[load dir="views" file="recent-posts.html"]

Auto-load fields

There are special fields that are automatically loaded into the page.

Head - CSS

Create a custom field named css, and the field will be included in the document head.

Load a page-specific stylesheet

[load css="landing-page.min.css"] By default, it looks for the stylesheet in the css folder of the current theme directory.

Load a stylesheet from a different location

[load dir="content" css="includes/css/font-awesome.min.css"]

Wrapping CSS script

[css]     .entry-content {         background-color: black;     } [/css]

Google Fonts

Use the gfonts parameter to include fonts from Google Fonts.

Include fonts and apply them to page elements

[load gfonts="Lato|Lora:400,700"] [css]     h1, h2 { font-family: Lora, serif; }     p { font-family: Lato, sans-serif; } [/css] This should be placed in the css field.

Foot - JS

Create a custom field named js, and the field will be included at the foot of document body.

Load a JavaScript file

[load js="slider.min.js"] By default, it looks for the file in the js folder of the current theme directory.

Wrapping JS script

[js]     jQuery(document).ready(function($){         ...     }); [/js] You can also load the script in the header, by placing the command in the css field.

HTML

Create a custom field named html, and the field will be displayed instead of the post content. This can be useful for pulling the post content into a layout.
