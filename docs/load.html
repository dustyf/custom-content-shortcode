Loading HTML, CSS and JS

Use the [load] shortcode to include a file. This could be useful for loading page-specific stylesheets, jQuery scripts, HTML templates, etc.

Header - CSS

Create a custom field called css, and the content of the field will be automatically added to the header of the page.

Load a stylesheet from the css field

[load css="bootstrap.min.css"]

Wrapping CSS script

[css]     .entry-content {         background-color: black;     } [/css]

Google Fonts

Use the [load] shortcode and the gfonts parameter to include fonts from Google Fonts.

Include fonts and apply them to page elements

[load gfonts="Lato|Lora:400,700"] [css]     h1, h2 { font-family: Lora, serif; }     p { font-family: Lato, sans-serif; } [/css] It should be placed in a custom field named css, so the fonts will be included in the header.

Footer - JS

Create a custom field called js, and the content of the field will be automatically added to the footer of the page, just before </body>.

Load a JavaScript file from the js field

[load js="bootstrap.min.js"]

Wrapping JS script

[js]     jQuery(window).load(function() {         jQuery('.isotope_container').isotope('reLayout');     }); [/js]

File location

By default, the [load] shortcode gets the specified file under the css or js folder in the theme directory. You can choose a different directory using the dir parameter.

Load a CSS file from a different location

[load dir="content" css="includes/css/font-awesome.min.css"] Available tags for the dir parameter:
  • web - http://
  • site - site address
  • wordpress - WordPress directory
  • content - /wp-content/
  • theme - theme directory - /wp-content/theme
  • child - child theme directory - /wp-content/child_theme

HTML

Create a custom field called html, and the content of the field will be displayed instead of the post content. This is could be useful for wrapping the post content in different layout templates. Use the [load] shortcode and the file parameter to include an HTML file. This can be done in the html field, or anywhere else. The file can contain shortcodes. [load dir="content" file="layout/sidebar.html"] If you don't set the dir parameter, it gets the file from the theme directory. To format the output with line breaks and paragraph tags, set the parameter format to true. If you want to disable shortcodes, set the parameter shortcode to false.
