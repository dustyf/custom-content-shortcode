Loading HTML/CSS/JS

Use the [load] shortcode to include a file into a page. This could be useful for loading page-specific stylesheets, jQuery scripts, layout templates, etc. For ease of use, create a field group called Scripts, with custom fields named html, css, and js. These are loaded into the content, header and footer of the page.

Header - CSS

Create a custom field called css, and the content of the field will be included in the header.

Load a stylesheet from the css field

[load css="bootstrap.min.css"]

Wrapping CSS script

[css]     .entry-content {         background-color: black;     } [/css]

Google Fonts

Use the [load] shortcode and the gfonts parameter to include fonts from Google Fonts.

Include fonts and apply them to page elements

[load gfonts="Lato|Lora:400,700"] [css]     h1, h2 { font-family: Lora, serif; }     p { font-family: Lato, sans-serif; } [/css] The code should be placed in a custom field named css, so the fonts will be included in the header.

Footer - JS

Create a custom field called js, and the content of the field will be added at the end of the page, just before </body>.

Load a JavaScript file

[load js="bootstrap.min.js"] You can also load JS files in the header by placing the above command in the css field.

Wrapping JS script

[js]     jQuery(window).load(function() {         jQuery('.isotope_container').isotope('reLayout');     }); [/js]

File location

By default, the [load] shortcode gets the specified file under the css or js folder in the theme directory. You can choose a different directory using the dir parameter.

Load a CSS file from a different location

[load dir="content" css="includes/css/font-awesome.min.css"] Available tags for the dir parameter:
  • web - http://
  • site - site address
  • wordpress - WordPress directory
  • content - /wp-content/
  • layout - /wp-content/layout/
  • theme - theme directory - /wp-content/theme
  • child - child theme directory - /wp-content/child_theme

HTML

Create a custom field called html, and the content of the field will be displayed instead of the post content. This is could be useful for wrapping the post content in different layouts, especially using a blank page template.

Include files

Use the [load] shortcode and the file parameter to include a file anywhere. The file can contain HTML, PHP script, and shortcodes. This can be used for creating flexible and reusable modules.

Load a sidebar

[load dir="layout" file="sidebar.html"] By default, the file parameter looks in the theme folder. To format the output with line breaks and paragraph tags, set the parameter format to true. If you want to disable PHP or shortcodes, set the parameter php or shortcode to false.
