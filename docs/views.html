Content views

Here is an example of how to create templates to display your content. Let's imagine a bicycle shop. We create a custom post type called bicycle, and add custom fields such as model, price, and description. Add all bicycles in the store as new entries, with featured image and other info in the fields. Create a new page to display the bicycles. A basic template could be: [loop type="bicycle"]     [content field="image"]     Model: [content field="model"]     Price: [content field="price"]     Description: [content field="description"] [/loop] Another page can display a list of bicycles in the freestyle category: Freestyle bikes available: <ul>     [loop type="bicycle" category="freestyle"]         <li>[content field="model"] - [content field="price"]</li>     [/loop] </ul>
