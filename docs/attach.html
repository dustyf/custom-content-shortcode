Attachments

Use theÂ [attached] shortcode to loop through attachments.

Display all attachments of the current post

[attached]     [field title]     [field image] or [content]     [field caption] [/attached] The following fields are available for each attachment.
  • id - ID
  • title - title
  • caption - caption
  • description - description
  • url - URL to attachment file
  • page-url - URL to attachment page
  • image - display image: <img src="~">
  • size - image size: thumbnail, medium, large, full (default) or custom defined size
  • thumbnail - thumbnail
  • thumbnail-url - thumbnail URL

Attachments of queried posts

Use [attached] inside a query loop to display attachments of specific posts.

Display attachment thumbnails of all posts in a category

[loop type="post" category="special"]     Post title: [field title]     [attached]         [field thumbnail]     [/attached] [/loop]

If post has attachment

Use [if attached] to display something if the post has any attachments. [loop type="post" category="special"]     Post title: [field title]     [if attached]         This post has attachments:         [attached][field thumbnail][/attached]     [/if]     [if not attached]         This post has no attachments.     [/if] [/loop]
